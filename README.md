# Swift-Zip-Manager
一个用 Swift 写的压缩文件管理器，极致简洁，仅 899 KB。

✨ 功能

· 解压 ZIP / TAR / GZ
· 支持 7z / RAR（通过安装工具）
· 创建 ZIP / TAR / GZ 归档文件
· 自定义解压路径
· 语言界面（14 种语言，正在开发中）
· 工具一键安装
· 帮助系统 + 快捷键支持

📱 平台

· macOS 13.5+

📦 体积:889 KB（比一张截图还小）

. v1.0.0-Beta.x ：基本完成，加一些小功能
. v1.0.0-RC.x ：最终验证
· v1.0.0（正式版）: 完整功能 + 宣传视频上线
