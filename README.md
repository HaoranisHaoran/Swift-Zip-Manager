# Swift-Zip-Manager
一个用 Swift 写的压缩文件管理器，极致简洁，仅 863 KB。

✨ 功能

· 解压 ZIP / TAR / GZ（使用系统自带命令）
· 支持 7z / RAR（通过 Homebrew 安装工具）
· 创建 ZIP / TAR / GZ 归档文件
· 自定义解压路径
· 语言界面占位（14 种语言，切换功能开发中）
· 工具安装引导（复制粘贴即可）
· 帮助系统 + 快捷键支持

📱 平台

· macOS 13.5+
